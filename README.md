# ai-chatbot
This is a basic web front end for an open-ai api
it is currently using GPT-4 to function as a grammar correction tool.

It is built using the official open-ai tools for javascript and node.js for the web-server.

This tool can be accessed locally after first starting the server
    start_server.bat 
    - or -
    start_server.sh
and following this link
    [link](http://localhost:3000/)
